Why LEDIT

Designed for real portable installs.

Skip fragile live sessions. LEDIT focuses on creating installed Linux systems that can boot from external storage with your selected distro, desktop, packages, firmware, and bootloader.

◎

One build model

GUI, TUI, and CLI normalize into the same configuration flow, so visual and terminal workflows stay consistent.

◈

Distro-native choices

Search official package repositories and choose release branches before adding packages to your image.

✦

Safer flashing

List removable drives, validate generated images, and flash only after confirming the target disk.

Workflow

From profile to bootable external system.

Run host checks, pick a distro, validate with dry-run, build the image, then flash it to a removable drive.

  1. 01Check host tools

    Run ./ledit doctor before building.

  2. 02Choose your target

    Select distro, release, packages, desktop, network, and firmware options.

  3. 03Dry-run and build

    Validate the plan first, then create the raw image.

  4. 04Flash carefully

    List devices and write the image only to the correct external drive.
